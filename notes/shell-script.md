# シェルスクリプト Memo

## 変数のデフォルト値（空だったらデフォルト値を入れる）

    # VAR が空なら "default value" をセット
    VAR=${VAR:="default value"}

- 参考： [Bashシェルの変数にデフォルト値を設定する - My Octopress Blog](http://buf-material.github.io/blog/2014/10/11/set-default-value-to-bash-variables/)
