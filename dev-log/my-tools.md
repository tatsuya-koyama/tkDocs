# 使ってるツール

## Mac 買い直したらこれ絶対入れるわーリスト

    - 不朽のエディタ。MacPorts 版を入れるとスクロールとかジェスチャーでできてリッチ

- [Spectacle](http://spectacleapp.com/)
    - ホットキーでウインドウをリサイズしたり移動したり
- [iTerm2](http://www.iterm2.com/#/section/home)
    - 標準 Terminal よりこちら
